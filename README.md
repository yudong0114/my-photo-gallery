# Yudong’s Photo Gallery
* 自分の撮影したサイトを作成

## 構成
* nuxt.js(フロント)
* microCMS(API)
* Github Actions(CI/CD)
* ロリポップ(サーバー)

## ビルド

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
